ANIMUS — START HERE
===================

Thank you for downloading ANIMUS.

What it is
----------
ANIMUS is a self-hosted AI workspace: a browser app you run on your own
computer, plus a bundled Hermes Agent source tree. You control where it runs;
your chats, projects, and API keys stay on your machine (not a vendor cloud).

Quick install (Linux or macOS)
------------------------------
1. Unzip this archive to a folder you own (for example ~/animus or ~/Apps/animus).

2. Open a terminal in that folder and run:

   chmod +x installer/install.sh installer/preflight.sh
   ./installer/preflight.sh
   ./installer/install.sh

3. Open http://127.0.0.1:3001 in your browser (default port; see animus.env.example).

4. Complete the on-screen setup wizard on first launch.
