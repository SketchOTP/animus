# Getting ANIMUS Updates

## Automatic updates
ANIMUS checks for updates automatically each time you open it.

When a new version is available, a banner appears at the top of the screen:
"ANIMUS v1.x.x is available — Update now / Later"

Click **Update now** to download and apply the update automatically.
Click **Restart now** when prompted. That's it.

## Manual check
Settings → Check for updates → checks right now instead of waiting.

## If an update fails
- Check your internet connection
- Try again from Settings → Check for updates
- If it keeps failing, re-download the latest version from your
  Gumroad Library at gumroad.com/library and reinstall.

## Re-downloading
Your purchase link always serves the latest version.
Go to gumroad.com/library, find ANIMUS, and download again.
Unzip into a new folder and run ./installer/install.sh.
Your data in ~/.hermes/chat is untouched by reinstall.
